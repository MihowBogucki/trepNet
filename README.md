# TrepNet Application

The Event Manager app is the showcased app in the free course: **Build your
first Firebase powered Ionic app** which covers:

* User Authentication.
* Profile Page.

## Installation

Clone this repository using ssh:

After project has been cloned:

```sh
$ npm install
```

Then 

```sh
$ ionic serve
```
